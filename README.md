## Разработка

### Инструменты

- React (18.1)
- Context API
- SCSS
- ESLint
- Material UI
- uuid

Для того, чтобы запустить приложение локально склонируйте репозиторий. После установите все зависимости при помощи `npm install`. Запустите приложение при помощи `npm run serve`.
